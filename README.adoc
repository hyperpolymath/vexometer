**The Vexometer: Irritation Surface Analyser (ISA) for LLMs and related tools** quantifies irritation across 10 dimensions with standardised, reproducible metrics. This monorepo contains ISA core, its satellite intervention system, the Lazy Eliminator completeness enforcer, and the Vext verifiable communications protocol.

== Naming

* *Canonical title*: *The Vexometer: Irritation Surface Analyser for LLMs and related tools*
* *Short name*: *ISA*
* The `vexometer` identifier is used in directory paths, package names, CLI commands, and protocol IDs (e.g. `vexometer-trace-v1`).
* See link:NAMING.md[NAMING.md] for the full naming policy.

NOTE: This repository was previously named `vex-tools`. The canonical name is now `vexometer`.

== Repository Structure

[cols="2,4", options="header"]
|===
| Directory | Description

| `vexometer/`
| **The Vexometer (ISA)** -- Irritation surface analyser for LLMs and related tools. Quantifies irritation across 10 dimensions with standardised, reproducible metrics.

| `vexometer-satellites/`
| **ISA Satellites (vexometer-satellites)** -- Umbrella documentation and coordination hub for the satellite intervention system.

| `satellite-template/`
| **Satellite Template** -- Scaffolding template for creating new ISA satellite projects.

| `lazy-eliminator/`
| **Lazy Eliminator** -- Completeness enforcement for LLM-generated code. Detects and eliminates lazy patterns.

| `vext/`
| **Vext** -- Verifiable, Extensible Communications Protocol with cryptographic proof of algorithmic neutrality.

| `vext-email-gateway/`
| **Vext Email Gateway** -- Email bridge for interoperability between Vext and traditional email (*prototype; end-to-end wiring in progress*).
|===
